# PracticeWork10M4 — Управління нерухомістю

## Опис проєкту

PracticeWork10M4 — це клієнтський WinUI 3 застосунок для керування об'єктами нерухомості. Програма реалізована з використанням архітектурного патерну MVVM та виконує HTTP-запити до ASP.NET Core Web API.

Застосунок дозволяє переглядати список об'єктів нерухомості, додавати нові записи, редагувати наявні дані та видаляти об'єкти через REST API.

## Тема практичної роботи

UWP/WinUI застосунок "Управління нерухомістю" з використанням архітектурного патерну MVVM, який створює HTTP-запити до веб-API.

## Функціональні можливості

- Завантаження списку об'єктів нерухомості з Web API.
- Відображення назви, адреси, власника, ціни оренди та статусу об'єкта.
- Додавання нового об'єкта нерухомості.
- Редагування вибраного об'єкта.
- Видалення об'єкта.
- Оновлення списку об'єктів.
- Перетворення числового статусу оренди у зрозумілий текстовий вигляд.

## Використані технології

- C#
- .NET 8
- WinUI 3
- XAML
- MVVM
- ASP.NET Core Web API
- HTTP-запити
- System.Net.Http.Json
- JSON

## Архітектура застосунку

Проєкт побудовано за архітектурним патерном MVVM:

- Model — описує структуру даних об'єкта нерухомості.
- View — XAML-інтерфейс користувача.
- ViewModel — містить логіку взаємодії між інтерфейсом та Web API.
- Service — виконує HTTP-запити до серверної частини.
